# 📖 About

**CodeCrack** is a modern Placement Preparation Platform built for engineering students to prepare for coding interviews, aptitude tests, resume optimization, company-specific interview preparation, and placement readiness—all from one application.

Instead of using multiple websites for coding practice, aptitude, resume analysis, interview preparation, and company research, CodeCrack combines everything into a single platform with a clean UI and real-time progress tracking.

# 📦 Installation

Clone repository

```bash
git clone https://github.com/G-Sabari/codecrack.git
```

Move into project

```bash
cd codecrack
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build project

```bash
npm run build
```
